# If you want to run it locally for some reasons

Download the repository, then using terminal, head in the repository directory and type :

<code> 
  npm install
</code> <br/>
<code>
  npm run dev
</code> <br/> <br/>


Now open your navigator and go to url : <code> localhost:1234 </code>

Enjoy !
